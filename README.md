Parsons.ai is an AI code editor for education.

We are building Parsons to help students learn programming by coding with AI, not just copying code. The editor combines hands-on coding, guided AI support, and learning checkpoints so learners can understand what is happening before changes are applied.

This repository contains the full source code for Parsons.ai. If you're new, welcome.

- 🧭 [Website](https://parsons.ai)

## Reference

Parsons is a fork of the [vscode](https://github.com/microsoft/vscode) repository. Fork is also a fork of Void. See [VOID_CODEBASE_GUIDE](https://github.com/voideditor/void/blob/main/VOID_CODEBASE_GUIDE.md) for Void.
